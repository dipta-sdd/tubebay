# Library Management

The **Library** is the central hub of TubeBay. It displays all the videos synchronized from your YouTube channel and provides the interface for mapping them to your WooCommerce products.

## The Video Grid
In the Library tab, you'll see a responsive grid of your YouTube videos. Each item includes:
- **Thumbnail:** The high-quality thumbnail from YouTube.
- **Title:** The video title as it appears on YouTube.
- **Mapping Status:** A badge indicating if the video is already mapped to a product.

## Searching Your Library
Use the search bar at the top of the Library to find specific videos by title. This search is performed against the local WordPress cache, making it instantaneous.

## Manual Sync
If you've recently uploaded a video to YouTube and it hasn't appeared yet, click the **Sync Library** button at the top right of the Library page to trigger an immediate update.

## Filtering
You can filter your library by:
- **All Videos:** View every synchronized video.
- **Mapped:** View only videos currently linked to products.
- **Unmapped:** Find videos that still need to be linked to your catalog.

## Bulk Actions (Pro)
*Note: Some features may require the Pro version.*
- **Bulk Map:** Select multiple videos and assign them to a category of products.
- **Auto-Map:** Attempt to match videos to products based on title similarity.
